# Project Alpha: Medieval Tower Defense x FPS Hybrid

## Overview
This project is a sandbox tower defense game featuring medieval themes, developed as a collaborative effort for **IT286** at NJIT. Players act as the kingdom's commander, strategically placing military forces to protect the castle from waves of enemies.

## Core Systems
* **Hybrid Perspective:** Players can remain in an omniscient commander view to regulate towers or switch directly into a troop's perspective for 1-to-many combat.
* **Economy & Strategy:** Access to various towers and power-ups is managed through a currency system tied to enemy kills.
* **Win/Loss Conditions:** If enemies breach the castle and health reaches zero, the game ends.
