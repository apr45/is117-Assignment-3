# AGENTS.md - Career Profile Web Strategy

## The Mandate
**Target Career Outcome:** Secure a summer internship as a Game Developer/Gameplay Programmer at an indie studio, AAA publisher, or mobile game company by leveraging a professional web portfolio that showcases game projects, digital art skills, and programming expertise. Secondary goals: build professional brand within NJIT community, attract collaborators for group game dev projects, and establish credibility for full-time game developer roles post-graduation.

**Timeline:** Portfolio + Flagship Game Project ready by Month 6 (early 2027) to align with 2027 summer internship recruiting season.

**Target Audience:**
- Game studio recruiters (primary)
- NJIT professors & peers (secondary)
- Indie game developers & collaborators (secondary)

---

## The Tech Stack

### Frontend & Website
**Technology: React + TypeScript + TailwindCSS**
- **Why:** Modern, game-dev-friendly portfolio (proves web skills), customizable, fast, and showcases technical depth
- **Alternative:** Next.js (adds SEO benefits via static generation, better for marketing-focused content)

### Portfolio Hosting
**Technology: Vercel or GitHub Pages**
- **Why:** Free, fast, Git-integrated, automatic deployments, excellent performance metrics for SEO

### Game Development Engine
**Primary: Unity (C#)**
- **Why:** Industry standard for indie and mid-tier studios, largest job market, excellent for 2D/3D games, strong networking in game dev community
- **Secondary: Unreal Engine (C++)** (if time permits) - Higher-end studios, more impressive on resume

### Content Management & Blog
**Technology: Markdown + Contentlayer or simple MDX**
- **Why:** Low overhead, integrates with React portfolio, version-controllable via Git, perfect for game dev devlogs

### Visual Assets & Design
**Tools:** Figma (portfolio design), Adobe Creative Suite (art assets), Blender (optional 3D assets)
- **Why:** Leverages your existing digital illustration skills, industry-standard for game asset creation

### Analytics & SEO Tools
**Technology: Google Analytics 4, Google Search Console, Lighthouse**
- **Why:** Free, actionable insights, essential for tracking internship lead generation

### Version Control & Deployment
**Technology: GitHub + GitHub Pages / Vercel**
- **Why:** Single source of truth, shows version control discipline, automated CI/CD, impressive to recruiters

## The Sitemap

### 1. Hero Section
**Purpose:** Immediate value proposition + hook recruiter attention

**Key Elements:**
- Headline: "Game Developer & Digital Artist | Python & Java | Unity Game Projects"
- Subheadline: "NJIT Student | Building Polished Games | Available for Summer 2027 Internships"
- CTA Button: "View My Games" (primary) | "GitHub" (secondary)
- Hero Image/Video: Animated hero showcasing a gameplay clip from your flagship game or game jam entry
- Social Proof: NJIT logo, potential game jam badges (once earned)

---

### 2. About Section
**Purpose:** Build credibility and showcase unique art + code combination

**Key Elements:**
- Professional photo (head shot)
- 150-word bio highlighting:
  - Art + Programming hybrid skill set (your competitive advantage)
  - Game development specialization
  - NJIT student status
  - 4 years of work experience (reliability)
- Skills breakdown:
  - **Programming:** Python, Java, C# (Unity)
  - **Game Engines:** Unity
  - **Art & Design:** Digital Illustration, Concept Art, UI/UX Design
  - **Soft Skills:** Problem-Solving, Teamwork, Critical Thinking, Creativity

---

### 3. Case Studies (Games Showcase)
**Purpose:** Proof of work; show shipped/playable projects

**Structure:** 3-5 project cards, each including:

#### Case Study Template:

**[Game Title]**
- **Genre:** [2D Platformer, Puzzle, Action, etc.]
- **Engine/Tools:** Unity, Python, Godot, etc.
- **Your Role:** Programmer, Artist, Designer, or hybrid
- **Problem Solved:** [What game mechanic/challenge did you tackle?]
- **Key Features:** [Bulleted list of 3-4 features]
- **Result/Impact:** [Lines played? GitHub stars? Game jam ranking?]
- **Visual:** Screenshot/GIF of gameplay
- **Links:** [Play Now] | [View Code on GitHub] | [Blog Post / Devlog]
- **Technologies:** Tag icons (Unity, C#, etc.)

#### Recommended Case Studies (by priority):

1. **Flagship Game (Priority 1 - In Development)**
   - Your main showcase project
   - Should be playable by internship application season
   - Showcase both code quality (GitHub) and visual polish (art skills)

2. **Game Jam Entry (Priority 2)**
   - Smaller scope, completed in 48-72 hours
   - Shows rapid iteration and deadline discipline
   - Great for showing you can *ship*

3. **University Project / Capstone (Priority 3)**
   - If applicable; shows academic rigor
   - Could be a game, game engine, or game tool

4. **Asset/Tool Showcase (Priority 4)**
   - Concept art, game sprites, or a game editor tool
   - Differentiates you from pure programmers

5. **Archived Learning Projects (Priority 5)**
   - Early projects to show progression
   - Keep brief; emphasize growth

### 4. Lead Generation / Contact Section
**Purpose:** Convert recruiter interest into direct contact

**Key Elements:**

**Form Fields (Minimal):**
- Full Name
- Email Address
- Company/Organization
- Message: "I'm interested in connecting about [internship / collaboration / etc.]"
- Submit Button: "Let's Connect" or "Schedule a Chat"

**Form Behavior:**
- Email confirmation sent to user
- Notification email to your inbox (alexandrap2005@outlook.com)
- Optional: Auto-responder email with portfolio link + GitHub
- Form storage: Google Sheets, Formspree, or Netlify Forms (no database required)

**Alternative CTAs:**
- Direct email: "Email Me" (email link)
- LinkedIn: "Connect on LinkedIn"
- GitHub: "Follow on GitHub"
- Discord/Socials: (if applicable for gaming community)

**Incentive (Optional):** "Download my Game Dev Resume" PDF or "Subscribe to my Game Dev Blog"
